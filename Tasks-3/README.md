# Task: Sales Prediction Using Python

## Objective
The objective of this task is to build a machine learning model that predicts product sales based on advertising expenditure across different platforms.

## Problem Statement
Sales prediction involves forecasting the amount of a product that customers will purchase, taking into account various factors such as advertising expenditure, target audience segmentation, and advertising platform selection.

## Dataset Features
The dataset may include:
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales (Target Variable)

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Data Preprocessing  
6. Train-Test Split  
7. Model Building (Linear Regression)  
8. Model Evaluation  
9. Sales Prediction  

## Machine Learning Model
Linear Regression model is used to predict sales based on advertising budgets.

## Goal
To build a machine learning model that predicts product sales and analyze the impact of advertising budgets on sales performance.

# Task 2: Movie Rating Prediction with Python

## Objective
The objective of this task is to build a machine learning model that predicts the rating of a movie based on features like genre, director, and actors.

## Problem Statement
Movie Rating Prediction project aims to analyze historical movie data and develop a model that accurately estimates the rating given to a movie by users or critics.

## Dataset Features
The dataset may include:
- Movie Name
- Genre
- Director
- Actors
- Duration
- Votes
- Year
- Rating

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Building (Regression)
7. Model Evaluation
8. Prediction

## Machine Learning Model
Regression techniques are used to predict movie ratings based on input features.

## Goal
To build a model that accurately estimates movie ratings and understand the factors that influence movie ratings.

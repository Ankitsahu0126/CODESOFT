# Task 1: Titanic Survival Prediction

## Objective
The objective of this task is to build a machine learning model that predicts whether a passenger survived the Titanic disaster or not.

## Dataset
Titanic Dataset containing passenger details like:
- Age
- Sex
- Pclass
- Fare
- SibSp
- Parch
- Embarked

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Building
6. Prediction
